# Vim-Cheatsheet

## Configure vim settings
```
vim ~/.vimrc
```
```
set expandtab
set tabstop=2
set shiftwidth=2
```

## Basic shortucts
```
i        Insert mode

```

## Command mode short cuts
```
esc      Command mode

dd       Delete line

:x       Exit and save changes if any
:q!      Exit without saving changes
```
