# py-venv
## Setup (Ensure you are already in project directory)
1. Create the venv
```
python3 -m venv venv
```

2. Activate
```
source venv/bin/activate
```

3. Install your dependencies
```
pip install --upgrade pip
pip install -r requirements.txt
```

4. Run script

5. When you're done
```
deactivate
```
